# 4d-tips-custom-focus
テキスト入力エリアのフォーカス表示をカスタマイズする例題

## 解説

*On Getting Focus* イベントでテキスト入力エリアの背景色を変更し，四角形オブジェクトを背後に表示しています。Mac版では面白いアニメーション効果がつきます。
